HTML
Hypertext Markup Language, the standard markup language for documents designed to be displayed in a web browser.
CSS
Cascading Style Sheets, a stylesheet language used for describing the presentation of a document written in HTML.
JavaScript
A programming language commonly used to create interactive effects within web browsers.
Python
Python is a high-level, interpreted, object-oriented programming language that's used in many areas, including: Web applications, Software development, Data science, Machine learning, Scripting automation, and Backend development
HTTP
Hypertext Transfer Protocol, the protocol used for transmitting web pages over the internet.
